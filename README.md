##  :gear: Run Application

1. Clone project

```
git clone git@github.com:ocarlosbarros/checkpoint.git
```

2. Access project folder

```
cd CheckPoint
```

3. Install dependencies

```
dotnet restore
```

4. Run project
```
dotnet run
```
